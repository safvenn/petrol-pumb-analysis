
# ⛽ Petrol Station Analytics Dashboard (Operations + HR)

## 📌 Project Overview  
This project presents an **end-to-end petrol station analytics solution** developed using **Power BI**, focusing on both **operational performance** and **HR workforce analytics**.  

The objective of this project is to transform large-scale transactional and employee data into **actionable business insights** that help optimize fuel sales, staffing, and operational efficiency.

The project consists of **two interactive dashboards**:  
- **Operations Dashboard** – Fuel sales, revenue, and customer activity  
- **HR Dashboard** – Employee productivity and workforce performance  

---

## 📊 Key Metrics & Scale  

- **136,550+ Transactions** analyzed  
- **22.06 Million Liters** of fuel sold  
- **₹2.15 Billion Total Revenue** generated  
- **25 Employees** performance tracked  
- **1 Million+ Working Hours** analyzed  
- **₹1.57 Million Monthly Salary** evaluated
