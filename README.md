
# ⛽ Petrol Station Analytics Dashboard
**End-to-End Data Analytics Project | Python → Power BI**

---

## 📌 Project Overview

This project demonstrates a complete **end-to-end data analytics workflow**, transforming messy petrol station datasets into meaningful business insights.

The solution includes:

- ⛽ Operations Analytics Dashboard  
- 👥 HR Analytics Dashboard  
- 🧹 Full Data Cleaning Pipeline  
- 📊 Exploratory Data Analysis  
- 🧮 KPI & Dashboard Development  

---

## 📊 Project Scale

- **150,000+ Transactions Processed**
- **22.06 Million Liters Analyzed**
- **₹2.15 Billion Revenue Evaluated**
- **25 Employees Performance Tracked**
- **1M+ Working Hours Analyzed**

---

## 🔄 End-to-End Workflow

### 📂 Step 1 — Data Collection

**Datasets Used:**

- transactions_pumb150k.csv  
- employee_pumb.csv  

**Data Includes:**

- Date & Time  
- Fuel Type  
- Liters Sold  
- Price Per Liter  
- Payment Method  
- Counter ID  
- Employee Details  
- Salary  
- Working Hours  

---

### 🧹 Step 2 — Data Cleaning & Preprocessing

**Tools Used:**

- Python  
- Pandas  
- NumPy  

**Cleaning Methods Applied:**

- Merged transaction and employee datasets  
- Removed duplicate records  
- Converted date columns to datetime format  
- Cleaned invalid numeric values  
- Handled missing values using mean and median  
- Recalculated Total Amount values  
- Replaced negative values with valid values  
- Standardized categorical fields  
- Applied IQR method for outlier detection  

---

**EDA Techniques:**

- Histogram analysis

- Outlier detection  
- Distribution analysis  

Analyzed:

- Fuel Sales Patterns  
- Price Trends

- Employee Data
